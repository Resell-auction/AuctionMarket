|===
| HTTP Status | Error Code | Message
| 500 Internal Server Error | BQ_QUERY_EXECUTION_FAILED | BigQuery 데이터 조회 중 오류가 발생했습니다.
| 500 Internal Server Error | BQ_RESULT_PROCESSING_FAILED | BigQuery 데이터 처리(결과 파싱) 중 오류가 발생했습니다.
| 500 Internal Server Error | BQ_OPERATION_INTERRUPTED | 데이터 조회 작업이 중단되었습니다.
| 500 Internal Server Error | BQ_JOB_FAILED | BigQuery 작업 실행에 실패했습니다.
| 500 Internal Server Error | BQ_JOB_NOT_FOUND | BigQuery 작업을 찾을 수 없습니다.
|===
